Marvel API

Information

The Marvel Comics API allow to access different resource types: Comics, Comic Series, Comic Stories, Comic Events, Creators and Characters.
It provides the abillity for searching and filtering the data by different aspects. All representations are encoded as JSON objects.
The API can be used in a various ways and help or improve those using apps or websites about the Marvel universe to access all the information easly.
